# 模型目录 model_catalog

> 供 `modelcrew-router`（判题型）与 `modelcrew-modeler`（选方法）查阅。
> 用法：先用"题目特征 → 模型族"速查表锁定候选族，再到对应小节看具体方法、Python 实现、变体命名。
> 命名变体很重要——竞赛里"加权 TOPSIS"比"TOPSIS"显得更有设计感。
> 检索方式（借鉴 MM-Agent 的 HMML 分层库）：**问题特征 → 模型族 → 具体方法**三层，先用速查表按关键词匹配族，再下钻到该族小节。

## 0. 题目特征 → 模型族 速查表（选型第一步）

| 题目里出现这种话 | 推荐模型族 |
|---|---|
| "求最优 / 如何分配 / 在约束下使…最大(小)" | **优化类**（LP/IP/NLP/启发式） |
| "预测 / 未来 / 趋势 / 时间序列" | **预测类**（回归/ARIMA/灰色/LSTM） |
| "评价 / 排名 / 综合得分 / 优劣" | **评价类**（AHP/TOPSIS/熵权/模糊综合） |
| "判断 / 分类 / 识别 / 归类" | **分类类**（Logistic/SVM/决策树/随机森林/NN） |
| "模拟 / 如果…会怎样 / 含随机性" | **仿真类**（蒙特卡罗/系统动力学/ABM） |
| "网络 / 路径 / 流量 / 连接" | **图论类**（最短路/最大流/最小生成树/PageRank） |
| "概率 / 分布 / 是否显著 / 相关" | **统计类**（描述统计/假设检验/方差分析/回归） |
| "随时间演化 / 增长 / 传播 / 动力学" | **动力系统类**（ODE/PDE/差分方程/SIR） |

> 多数真题是**复合型**：先按子问题分别定族，再组合。

## 1. 优化类 Optimization
| 方法 | 适用 | Python | 变体名 |
|---|---|---|---|
| 线性规划 LP | 目标+约束都线性 | `scipy.optimize.linprog`, `pulp`, `cvxpy` | 多目标加权 LP、鲁棒 LP |
| 整数/0-1 规划 IP | 决策变量取整/0-1 | `pulp`, `cvxpy`+CBC/GUROBI | 分支定界 MIP、Lagrangian 松弛 |
| 非线性规划 NLP | 含非线性项 | `scipy.optimize.minimize`(SLSQP) | 凸近似、二阶锥 SOCP |
| 多目标优化 | 多个冲突目标 | `pymoo`(NSGA-II) | Pareto 前沿、ε-约束法 |
| 启发式 | 大规模/非凸/组合爆炸 | `scipy`, 手写 | 遗传算法、模拟退火、粒子群 PSO、蚁群 |

## 2. 预测类 Prediction / 时间序列
| 方法 | 适用 | Python | 变体名 |
|---|---|---|---|
| 多元回归 | 因变量~多自变量、要可解释 | `statsmodels`, `sklearn` | 岭回归、Lasso、逐步回归 |
| ARIMA / SARIMA | 平稳/季节性时间序列 | `statsmodels` | 带外生变量 ARIMAX |
| 灰色预测 GM(1,1) | **小样本**趋势（国赛常用） | 手写/`pyGM` | 残差修正 GM、Verhulst |
| 指数平滑 | 带趋势/季节 | `statsmodels`(ETS) | Holt-Winters |
| LSTM / GRU | 长序列、非线性、数据量够 | `pytorch`, `keras` | 注意力 LSTM、Seq2Seq |

## 3. 评价类 Evaluation / 综合评价
| 方法 | 适用 | 要点 | 变体名 |
|---|---|---|---|
| 层次分析法 AHP | 主观权重、有层次结构 | 一致性比 CR<0.1 | 群决策 AHP、模糊 AHP |
| 熵权法 | **客观**权重（数据定权） | 信息熵越小权重越大 | 与 AHP 组合的"主客观组合赋权" |
| TOPSIS | 多方案排序、距理想解远近 | 先归一化 | 熵权-TOPSIS（最常见组合） |
| 模糊综合评价 | 评价边界模糊（优良中差） | 隶属度函数 | 多级模糊评价 |
| 灰色关联分析 GRA | 小样本、找主导因素 | 关联度排序 | 灰色-TOPSIS |

## 4. 分类/判别类 Classification
| 方法 | 适用 | Python | 备注 |
|---|---|---|---|
| Logistic 回归 | 二分类、要可解释 | `sklearn` | 输出概率、看系数 |
| SVM | 中小样本、高维 | `sklearn` | 核技巧处理非线性 |
| 决策树 / 随机森林 | 非线性、要特征重要性 | `sklearn` | RF 抗过拟合、给重要性排序 |
| XGBoost / LightGBM | 结构化数据竞赛王者 | `xgboost`,`lightgbm` | 注意调参与过拟合 |
| 神经网络 | 数据量大、特征复杂 | `pytorch` | 小数据慎用 |
| 聚类 KMeans/层次/DBSCAN | **无标签**分群 | `sklearn` | 先定 k（肘部法/轮廓系数） |

## 5. 仿真类 Simulation
| 方法 | 适用 | 要点 |
|---|---|---|
| 蒙特卡罗 | 含随机性、解析难 | 大量随机抽样求期望/分布；配拉丁超立方抽样 |
| 系统动力学 SD | 反馈回路、存量-流量 | `pysd`、因果回路图 |
| 元胞自动机 / ABM | 个体交互涌现宏观 | 传染、交通、人群 |
| 排队论 | 服务/等待系统 | M/M/1、M/M/c |

## 6. 图论/网络类 Graph
| 方法 | 适用 | Python |
|---|---|---|
| 最短路 Dijkstra/Floyd | 路径最短 | `networkx` |
| 最大流/最小割 | 流量/瓶颈 | `networkx` |
| 最小生成树 | 连通最省 | `networkx`(Kruskal/Prim) |
| 网络中心性/PageRank | 节点重要性 | `networkx` |
| TSP/VRP | 旅行商/车辆路径 | `ortools`, 启发式 |

## 7. 统计类 Statistics
| 方法 | 适用 |
|---|---|
| 描述统计 + 可视化 | 任何数据题的第一步 |
| 假设检验（t/卡方/秩和） | 判断差异是否显著 |
| 方差分析 ANOVA | 多组均值比较 |
| 相关分析（Pearson/Spearman） | 量化关联（注意≠因果） |
| 主成分分析 PCA | 降维、去相关 |

## 8. 动力系统类 Dynamical Systems
| 方法 | 适用 | Python |
|---|---|---|
| 常微分方程 ODE | 连续随时间演化 | `scipy.integrate.solve_ivp` |
| 偏微分方程 PDE | 时空演化（扩散/热传导） | `py-pde`, 有限差分 |
| 差分方程 | 离散时间递推 | 手写迭代 |
| 传染病模型 SIR/SEIR | 传播过程 | `scipy` 解 ODE |

---
> 选型铁律（交给 Modeler）：**先选最简单能解释的方法做 baseline，再上复杂模型并与 baseline 对比**。复杂≠高分，能说清"为什么这个方法"才高分。
