# Round 02 — 数据型：某市共享单车站点日租车量预测与特征分析

## 背景

某市 2024 年 6 月运营 20 个共享单车站点，记录了各站点特征和日均租车量。需建立预测模型，识别关键影响因子，并发现异常站点。

**变量说明**：
- 人口密度：站点周边 1km² 内常住人口（千人）
- 距地铁：到最近地铁站的步行距离
- 自行车道：站点 500m 半径内自行车道总长度
- POI 数：500m 内兴趣点（商铺、学校、办公楼等）
- 办公/居住比：周边办公面积与居住面积之比
- 气温：当月日均气温
- 日租车量：日均租出次数（目标变量）

## 子问题

1. **预测模型**：建立日租车量的多元回归模型，报告 R²、RMSE 和 5 折交叉验证 RMSE。
2. **特征重要性**：哪些特征对租车量影响最大？用标准化系数和逐步回归分析排名。
3. **异常站点**：哪些站点的实际租车量显著偏离模型预测？分析可能原因。
