# R2 Model — 共享单车需求预测

## 候选方法
| # | 方法 | 优点 | 缺点 |
|---|------|------|------|
| M1 | OLS多元回归 | 经典、可解释 | 对共线性敏感 |
| M2 | Lasso回归 | 自动特征选择、正则化 | n=20时选择可能不稳定 |
| M3 | 随机森林 | 非线性、交互 | n=20太小 |

## 选型: M1(OLS) + M2(Lasso)对比

## 待审计结论
C1: R²和RMSE, C2: 5-fold CV RMSE, C3: 标准化系数排名
C4: VIF共线性, C5: 异常站点识别, C6: 逐步回归选择结果
