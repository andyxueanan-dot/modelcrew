# Problem Analysis — 派单策略 A/B 评估

## ⚠️ 辛普森悖论检测

本题存在**教科书级辛普森悖论 (Simpson's Paradox)**：

| 视角 | 策略 A | 策略 B | 谁更好？ |
|------|--------|--------|---------|
| **汇总** | 290/350 = 82.9% | 182/350 = 52.0% | A 赢 30.9pp |
| **低难度层** | 45/50 = 90.0% | 282/300 = 94.0% | **B 赢 4.0pp** |
| **高难度层** | 245/300 = 81.7% | 20/50 = 40.0% | **B 赢 41.7pp** |

汇总看 A 赢，但每一层都是 B 更好。原因是**混杂变量：订单难度**。
- A 的 300/350 (85.7%) 订单是高难度的
- B 的 300/350 (85.7%) 订单是低难度的
- A 被分配了更多难单，导致汇总率被拉低

## Sub-Questions

| ID | Restatement | Type | Priority |
|----|-----------|------|----------|
| **Q1** | 到底哪套策略更优？需要同时从汇总和分层两个视角分析，给出统计检验 + 因果论证 | Required (KEY) | ⭐⭐⭐ |
| **Q2** | 若要全市推广，给建议 + 风险分析 | Required | ⭐⭐ |
| **Q3** | 结论对分层口径（怎样定义"高/低难度"）有多敏感？ | Required | ⭐⭐ |

## Key Definitions

| Term | Definition | Source |
|------|-----------|--------|
| **按时送达率** | 按时送达订单数 / 总订单数 × 100% | 题目定义 |
| **辛普森悖论** | 汇总数据中的趋势在分层后消失或反转的现象 | 统计学经典概念 |
| **混杂变量 (Confounder)** | 同时影响分组分配和结果的变量；此处为"订单难度" | 因果推断 |
| **"更优"的定义** | 需要明确：是在当前订单分配下的整体表现？还是在标准化（等比例难度分配）后的表现？ | **我们的假设——这是本题核心争议** |
| **Mantel-Haenszel 估计** | 分层后合并的共同效应量估计，可控制混杂变量 | 统计方法 |

## Assumptions

| # | Assumption | Why | Fragility |
|---|-----------|-----|-----------|
| A1 | 订单难度只有"低"和"高"两个类别，且分类准确 | 题目给定的分层口径 | 现实中难度是连续谱；二分法可能过度简化。**Q3 正是要审计这一点** |
| A2 | 350 单的策略 A 和策略 B 是独立随机样本 | 标准统计假设 | 如果同一时段的订单存在相关性（天气、交通），独立性被违反 |
| A3 | 策略 A 和 B 的"高难度"订单具有可比性 | 分层分析的前提 | 如果 A 和 B 对"高难度"的定义不同（A 的"高难度"比 B 的更难），分层比较无效 |
| A4 | 样本量足够支撑分层分析 | 每层 n≥50 | 最小层 B-高难度 n=50，borderline；置信区间会较宽 |
| A5 | 推广到全市时，订单难度分布可能与测试期不同 | 外推需要 | 如果全市的难/易比例变化，最优策略可能反转 |

## Traps & Warnings

1. **🚨 辛普森悖论陷阱**：最诱人的错误是只看汇总数据就宣称"A 更优"。汇总的 82.9% vs 52.0% 差距巨大，看起来很确定——但这完全是因为订单分配不均造成的。

2. **"更优"的定义不唯一**：
   - 如果"更优" = 在当前分配下的整体按时率 → A 确实更好 (82.9% > 52.0%)
   - 如果"更优" = 在相同难度条件下的表现 → B 在两层都更好
   - 如果"更优" = 标准化后的加权率 → 需要计算 Mantel-Haenszel 或标准化率
   - **论文必须明确定义"更优"，并讨论不同定义下的不同结论**

3. **因果 vs 相关**：策略 → 订单难度分配 → 按时率。如果我们能控制分配（如全市推广时统一分配规则），那"策略本身的能力"才是关键。如果不能控制，那汇总数据可能更反映实际部署效果。

4. **小样本层**：B-高难度层只有 50 单（20 送达 = 40%），置信区间极宽（95% CI ≈ [26%, 55%]）。B 在高难度层的"40%"可能不够精确。

5. **Q3 敏感性**：如果将"难度"重新定义为 3 级或连续变量，结论是否改变？需要讨论。
