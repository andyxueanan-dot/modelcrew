# R1 Model — 研发项目组合优选

## 候选方法

| # | 方法 | 优点 | 缺点 |
|---|------|------|------|
| M1 | **0-1 IP 枚举法** | 2^6=64 组合，完全枚举保证全局最优 | 仅适用于小规模 |
| M2 | **整数规划 (scipy.linprog/PuLP)** | 通用求解器，可扩展 | 需要额外安装 |
| M3 | **动态规划 (多维背包)** | 理论优雅，可扩展 | 状态空间大 |

## 选型：M1（完全枚举）

理由：n=6 时 2^6=64 种组合完全可枚举，保证找到全局最优且便于敏感性分析。Q2 和 Q3 也可在同框架下修改约束条件重新枚举。

## 数学模型

**决策变量**：x_i ∈ {0,1}, i ∈ {A,B,C,D,E,F}

**Q1 目标函数**：max Σ r_i · x_i

**约束**：
- Σ c_i · x_i ≤ 150 （预算）
- Σ s_i · x_i ≤ 40 （人力）
- x_D ≤ x_B （依赖）

**Q2**：min Σ c_i · x_i  s.t. Σ x_i ≥ 3, Σ r_i · x_i ≥ 350, x_D ≤ x_B

**Q3**：同 Q1，但 r_C = 280
