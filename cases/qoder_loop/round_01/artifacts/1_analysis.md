# R1 Analysis — 农业种植方案优化

## Sub-Questions
| ID | Restatement | Priority |
|----|-----------|----------|
| Q1 | 在用水/化肥/粮食产量约束下最大化净利润 | ⭐⭐⭐ |
| Q2 | 利润增加10%所需的最小额外用水配额(影子价格) | ⭐⭐ |
| Q3 | 棉花售价跌至10元/kg后最优方案是否改变 | ⭐⭐ |

## Key Definitions
- **净利润**: 总销售收入 - 总种子成本 (本题中种子成本是唯一显式成本)
- **土壤适配**: 地块土壤质量指数 >= 作物最低土壤要求
- **实际用水**: 受灌溉方式调整——滴灌×1.0, 漫灌÷0.8=×1.25, 雨养×0.4

## Assumptions
A1. 每块地只种一种作物, 不混种
A2. 产量和价格为确定性参数
A3. 灌溉方式对产量的影响已包含在产量参数中
A4. 不考虑轮作历史(单期决策)

## Traps
1. 漫灌效率: 用水参数÷0.8, 意味着漫灌比滴灌多耗水25%
2. 雨养减免: 地块3只计40%用水, 可能使雨养地块种高水耗作物成为可行
3. 粮食产量约束: 只计小麦+玉米+大豆, 棉花不算粮食
4. 土壤适配: 地块3(0.60)不能种棉花(要求0.65)
