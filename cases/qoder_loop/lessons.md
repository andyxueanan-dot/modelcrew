# Lessons

## R1 (优化型·农业种植) — 7.19 / M
**最值钱教训**: 出题时约束参数的松紧度直接决定论文分析深度。粮食约束(≥80000kg)实际产出210000kg,松弛162%导致约束几乎无效。出题时应先跑一遍solver校准约束参数,确保至少一条约束接近紧致。

## R2 (数据型·共享单车) — 7.50 / M
**最值钱教训**: 出题时特征之间的相关性需要预先设计,否则会陷入"R2极高但0个显著特征"的极端共线性困境。本题5个"城市密度"类特征(PopDensity/POI/BikeLane/OfficeRes)本质测同一构念,VIF最高达341。虽然论文正确诊断了共线性,但**缺少PCA这一最自然的解决方案**成为最大扣分点。数据题应在solver中预埋PCA/PCR作为共线性解决方案,即使主模型不做PCA,也应在Discussion中展示。

## R3 (评价型·宜居城市) — 7.42 / M
**最值钱教训**: 评价题的样本量(n)直接决定方法论的可信度。本题n=8城市太小,导致(1)排名对数据扰动敏感,缺乏Bootstrap重采样检验;(2)指标间相关性无法有效分析(n太小相关矩阵不可靠)。评价题应设计n≥15的样本量,并在solver中预埋Bootstrap/Jackknife重采样检验排名稳定性(95% CI)。此外,多方法MCDM的敏感性分析不能只停留在"两种方法排名相关",必须测试关键权重(如GDP权重)变化导致的排名反转。
