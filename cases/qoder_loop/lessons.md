# Lessons

## R1 (优化型·农业种植) — 7.19 / M

**最值钱教训**: 出题时约束参数的松紧度直接决定论文分析深度。本题粮食约束(≥80000kg)实际产出210000kg,松弛162%导致约束几乎无效——评委虽认可"发现松弛"是诚实的分析,但更希望看到"约束收紧到何处时最优方案改变"的敏感性扫描。**出题时应先跑一遍solver校准约束参数**,确保至少一条约束接近紧致,才有足够的分析空间。

**其他教训**:
- "棉花悖论"(个体利润最高但系统不选)是论文最大亮点,评委一致好评——组合优化题应有此类反直觉发现
- Q2扫描步长1000太粗,应在粗扫后做二分法精确定位
- LP松弛上界是优化题的低成本高回报补充(5行代码)
