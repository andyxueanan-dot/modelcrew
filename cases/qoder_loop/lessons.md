# Lessons — 无限自练沉淀

## R1 (优化型·项目组合优选) — 6.87 / H

**最值钱教训**：题目给了风险等级数据但模型完全忽略了它——这直接导致"结果分析"和"模型选择"两个维度被扣分。教训：**题目中给出的每一个数据维度都应在模型中体现（量化或至少定性讨论），否则评委会认为分析不完整。** 具体做法：即使主模型不纳入风险，也应在 Discussion 中做一个简单的风险加权敏感性分析（3 行代码），展示"如果考虑风险，最优组合是否改变"。

**其他教训**：
- 对 n=6 的简单问题，枚举法正确但缺乏方法论深度——应补充 LP 松弛上界分析和计算复杂度讨论
- Shadow price 用离散增量（+5）估算时，必须标注"非连续对偶价格"
- 图表要有信息增量：图 3（Q1 vs Q3 同组合不同数字）信息量低，应替换为约束边界可视化
