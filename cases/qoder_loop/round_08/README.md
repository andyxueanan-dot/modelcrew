# Round 08 — 数据型（第 2 轮）

## 题目：二手房价格预测——多重共线性与主成分回归

某市 50 套二手房的交易数据，包含 8 个特征变量和 1 个目标变量（成交单价，万元/㎡）。特征变量之间存在明显的多重共线性（如面积与房间数、楼层与总楼层等）。

### 数据特征

| 特征 | 符号 | 单位 | 说明 |
|------|------|------|------|
| 建筑面积 | X1 | ㎡ | 50-200 |
| 房间数 | X2 | 间 | 1-6 |
| 楼层 | X3 | 层 | 1-30 |
| 总楼层 | X4 | 层 | 6-35 |
| 建筑年代 | X5 | 年 | 1990-2020 |
| 地铁距离 | X6 | km | 0.1-5.0 |
| 学区评分 | X7 | 分 | 1-10 |
| 商圈评分 | X8 | 分 | 1-10 |
| **成交单价** | **Y** | **万元/㎡** | **2.0-12.0** |

### 已知共线性结构

- X1（面积）与 X2（房间数）高度相关（r ≈ 0.85）
- X3（楼层）与 X4（总楼层）中度相关（r ≈ 0.60）
- X7（学区）与 X8（商圈）中度相关（r ≈ 0.55）

### 问题

**Q1**: 用 OLS 回归拟合全模型（8 个特征），R² 是多少？哪些特征显著（p<0.05）？VIF（方差膨胀因子）是否超过 10？

**Q2**: 如果存在严重共线性（VIF>10），用主成分回归（PCR）降维。需要保留多少个主成分才能解释 90% 的方差？PCR 的预测精度（RMSE）与 OLS 相比如何？

**Q3**: 用逐步回归（前向选择）筛选特征。最终保留哪些特征？模型的 R² 和 RMSE 是多少？

### 要求

1. 使用 numpy 生成合成数据（seed=42）
2. 计算相关矩阵和 VIF
3. OLS 回归 + 显著性检验
4. 主成分分析（PCA）+ 主成分回归（PCR）
5. 逐步回归（前向选择）
6. 5 折交叉验证比较 OLS、PCR、逐步回归的预测精度
7. 可视化：相关矩阵热力图、PCA 碎石图、预测精度对比
