# Analysis — 二手房价格预测与共线性处理

## 1. 问题分解

### Q1: OLS 全模型诊断
- **核心**: 拟合 8 特征 OLS 回归，诊断多重共线性
- **输出**: R², 系数显著性, VIF

### Q2: 主成分回归 (PCR)
- **核心**: PCA 降维 + PCR 回归，解决共线性
- **输出**: 主成分数、方差解释比例、PCR 的 RMSE

### Q3: 逐步回归
- **核心**: 前向选择筛选特征，简化模型
- **输出**: 保留的特征、R²、RMSE

## 2. 假设清单

| # | 假设 | 理由 |
|---|------|------|
| A1 | 数据是独立同分布的随机样本 | 标准回归假设 |
| A2 | 误差项正态分布且同方差 | OLS 显著性检验前提 |
| A3 | 特征与目标变量线性相关 | 线性回归模型假设 |
| A4 | 共线性是结构性的（非偶然） | 面积与房间数天然相关 |
| A5 | 合成数据的共线性结构反映真实场景 | seed=42 保证可复现 |

## 3. 数据生成策略（R2 教训应用）

### 共线性设计
- X1（面积）= 基础变量 + 噪声
- X2（房间数）= 0.8×X1 + 噪声（r≈0.85）
- X3（楼层）= 基础变量
- X4（总楼层）= 0.6×X3 + 噪声（r≈0.60）
- X7（学区）= 基础变量
- X8（商圈）= 0.55×X7 + 噪声（r≈0.55）

### 目标变量
Y = 2 + 0.03×X1 + 0.5×X2 - 0.1×X3 + 0.2×X5 - 0.8×X6 + 0.6×X7 + 噪声

## 4. 预期陷阱

### 陷阱 1: VIF 爆表
X1 和 X2 的高度共线性可能导致 VIF > 10，使得 OLS 系数不显著。

### 陷阱 2: PCR 的组件选择
保留太少主成分会丢失信息，保留太多会重新引入共线性。需要碎石图辅助决策。

### 陷阱 3: 逐步回归的稳定性
共线性下，逐步回归的选择结果可能对数据扰动敏感（bootstrap 验证）。

## 5. 方法选择理由

### 为什么用 PCR？
- PCR 是解决多重共线性的标准方法
- 通过降维消除共线性，同时保留大部分方差
- 比岭回归更直观（主成分可解释）

### 为什么用逐步回归？
- 逐步回归是特征筛选的标准方法
- 可以识别"真正重要"的特征
- 与 PCR 形成对比（降维 vs 选择）

### 为什么用 5 折交叉验证？
- 避免过拟合
- 给出更可靠的预测精度估计
- 三种模型在相同数据上对比，公平比较
