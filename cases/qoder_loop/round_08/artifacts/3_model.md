# Model — 多重共线性与主成分回归

## 1. 数据模型

### 特征变量
X ∈ ℝ^(50×8): 8 个特征，50 个样本

### 目标变量
Y ∈ ℝ^50: 成交单价（万元/㎡）

### 真实关系
Y = β₀ + β₁X₁ + β₂X₂ + ... + β₈X₈ + ε

其中 ε ~ N(0, σ²)

## 2. OLS 回归

### 模型
Ŷ = Xβ̂, β̂ = (X^T X)^(-1) X^T Y

### 诊断指标
- R² = 1 - SS_res / SS_tot
- 调整 R² = 1 - (1-R²)(n-1)/(n-p-1)
- 系数 t 检验: t_j = β̂_j / SE(β̂_j)
- VIF_j = 1 / (1 - R²_j), 其中 R²_j 是 X_j 对其他特征的回归 R²

### 共线性判断
- VIF > 10: 严重共线性
- VIF > 5: 中度共线性

## 3. 主成分分析 (PCA)

### 标准化
X_std = (X - μ) / σ

### 协方差矩阵
C = X_std^T X_std / (n-1)

### 特征分解
C = VΛV^T

其中 Λ = diag(λ₁, λ₂, ..., λ₈), λ₁ ≥ λ₂ ≥ ... ≥ λ₈

### 主成分
Z_k = X_std · v_k, k = 1, 2, ..., K

### 方差解释比例
PVE_k = λ_k / Σ λ_j

### 累积方差解释
CVE_K = Σ_{k=1}^K PVE_k

### 组件选择
选择最小的 K 使得 CVE_K ≥ 0.90（90% 方差解释）

## 4. 主成分回归 (PCR)

### 模型
Ŷ = Z_K γ̂, γ̂ = (Z_K^T Z_K)^(-1) Z_K^T Y

### 预测
Ŷ_new = (X_new - μ) / σ · V_K · γ̂

## 5. 逐步回归（前向选择）

### 算法
```
1. 初始化: 模型为空，特征集合 S = {}
2. For each feature j not in S:
   a. 拟合模型 S ∪ {j}
   b. 计算 AIC 或调整 R²
3. 选择使 AIC 最小（或调整 R² 最大）的特征 j*
4. If AIC 改善（或 p-value < 0.05）:
   a. S = S ∪ {j*}
   b. Go to 2
5. Else: 停止
```

## 6. 交叉验证

### 5 折 CV
```
1. 将数据分为 5 折
2. For each fold k:
   a. 用 4 折训练模型
   b. 在第 k 折上预测
   c. 计算 RMSE_k
3. CV-RMSE = mean(RMSE_k)
```

## 7. 输出指标

| 指标 | 公式 | 含义 |
|------|------|------|
| R² | 1 - SS_res/SS_tot | 拟合优度 |
| 调整 R² | 1 - (1-R²)(n-1)/(n-p-1) | 惩罚复杂度 |
| RMSE | √(Σ(Y-Ŷ)²/n) | 预测误差 |
| VIF_j | 1/(1-R²_j) | 共线性程度 |
| PVE_k | λ_k/Σλ_j | 主成分方差解释 |
| CVE_K | Σ PVE_k | 累积方差解释 |
