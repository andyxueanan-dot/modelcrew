# Results — 二手房价格回归分析

## 1. 相关性分析

### 相关矩阵（关键对）

| 特征对 | Pearson r | 共线性程度 |
|--------|-----------|-----------|
| X1(面积) - X2(房间数) | 0.954 | 严重 |
| X3(楼层) - X4(总楼层) | 0.772 | 较强 |
| X7(学区) - X8(商圈) | 0.832 | 较强 |

设计共线性结构被成功复现：X1-X2 超过预设 0.85，X7-X8 超过预设 0.55。

## 2. OLS 全模型

### 拟合优度

| 指标 | 值 |
|------|-----|
| R² | 0.9433 |
| 调整 R² | 0.9322 |
| F 统计量 | 85.36 |
| F p-value | 1.24e-24 |

### 系数显著性（p<0.05）

5/8 个特征显著：X3(楼层)、X5(建筑年代)、X6(地铁距离)、X7(学区评分)、X8(商圈评分)。

不显著：X1(面积, p=0.32)、X2(房间数, p=0.18)、X4(总楼层, p=0.41)。

### VIF 诊断

| 特征 | VIF | 判断 |
|------|-----|------|
| X1(面积) | 13.80 | >10 严重 |
| X2(房间数) | 14.48 | >10 严重 |
| X3(楼层) | 3.21 | <5 安全 |
| X4(总楼层) | 2.87 | <5 安全 |
| X5(建筑年代) | 1.52 | <5 安全 |
| X6(地铁距离) | 1.89 | <5 安全 |
| X7(学区评分) | 4.67 | <5 安全 |
| X8(商圈评分) | 3.95 | <5 安全 |

**结论**：X1 和 X2 存在严重共线性（VIF>10），与高相关系数 r=0.954 一致。OLS 虽然 R²=0.9433 很高，但面积和房间数都不显著——经典的"模型整体显著但个体不显著"共线性症状。

## 3. 主成分分析 (PCA)

### 碎石图数据

| 主成分 | PVE | 累积 CVE |
|--------|-----|---------|
| PC1 | 0.281 | 0.281 |
| PC2 | 0.233 | 0.514 |
| PC3 | 0.157 | 0.671 |
| PC4 | 0.119 | 0.790 |
| PC5 | 0.091 | 0.881 |
| PC6 | 0.052 | 0.933 |
| PC7 | 0.038 | 0.971 |
| PC8 | 0.029 | 1.000 |

**选择 K=5**（CVE=0.9486 > 0.90 阈值），但 K=5 时 CVE=0.881 < 0.90，实际需要 **K=6** 才能超过 90%。

修正：K=6 个主成分，CVE=0.933 > 0.90。

## 4. 主成分回归 (PCR)

| 指标 | 值 |
|------|-----|
| 主成分数 K | 6 |
| R² | 0.9222 |
| 调整 R² | 0.9113 |

PCR 的 R² 比 OLS 低 2.1 个百分点（0.9433 → 0.9222），这是降维的代价：丢弃了 2 个主成分中包含的部分信号。但 PCR 的优势在于系数稳定性——消除了共线性导致的系数大幅波动。

## 5. 逐步回归（前向选择）

### 入选特征（按进入顺序）

1. X7(学区评分) — 单独解释力最强
2. X5(建筑年代) — 年代越新价格越高
3. X1(面积) — 面积效应
4. X6(地铁距离) — 交通可达性
5. X3(楼层) — 楼层效应
6. X2(房间数) — 边际贡献显著时加入

### 模型性能

| 指标 | 值 |
|------|-----|
| 保留特征数 | 6 |
| R² | 0.9430 |
| 调整 R² | 0.9351 |

前向选择剔除了 X4(总楼层) 和 X8(商圈评分)，R² 几乎不变（0.9433 → 0.9430），调整 R² 反而提升（0.9322 → 0.9351），说明简约模型更优。

## 6. 五折交叉验证

| 模型 | CV-RMSE | 排名 |
|------|---------|------|
| 前向选择 | 0.6729 | 1 |
| OLS 全模型 | 0.6784 | 2 |
| PCR (K=6) | 0.7581 | 3 |

**结论**：前向选择以微弱优势胜出（RMSE 比 OLS 低 0.8%），说明剔除 2 个冗余特征减少了过拟合。PCR 的 CV-RMSE 最高，因为降维丢弃了有用信息——在 n=50、p=8 的场景中，共线性虽存在但不极端到需要 PCA 的程度。

## 7. 核心发现

1. **共线性诊断有效**：VIF 准确识别了 X1-X2 共线性对（VIF 均>10），与相关矩阵一致。
2. **"高 R² 低显著"症状**：OLS R²=0.9433 但 3/8 特征不显著，是共线性导致标准误膨胀的典型表现。
3. **前向选择最优**：6 特征模型在 CV 中胜出，调整 R² 也最高，说明 parsimony 原则在中等共线性下优于降维。
4. **PCR 的适用边界**：当共线性不极端（最大 VIF=14.48 而非 R2 题中的 341）且样本量适中时，特征选择优于主成分降维。
