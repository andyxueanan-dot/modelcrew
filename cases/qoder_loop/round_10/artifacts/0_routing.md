# Routing — 城市应急疏散网络

## 题型判定
**类型**: 图论网络型 (Graph/Network)

## 路由理由
1. 有向图最短路径 + 容量约束问题
2. 需要多种图论算法：Dijkstra + Yen's k-shortest + Edmonds-Karp max flow + connectivity
3. 含节点失效分析和连通度计算
4. 吸取 R4 教训：预埋 k-最短路径、κ/λ、多节点失效

## 核心挑战
1. **"等时异容"悖论**: 三条最短路径时间相同但容量不同
2. **方向性割点**: M2 在有向图中是关键节点，但无向图中不是割点
3. **S3→T1 结构性脆弱**: 边连通度仅 1

## 方法链
```
Dijkstra(9对S→T) → Yen's K=3(S1→T1) → Edmonds-Karp最大流
→ κ/λ连通度 → 单节点失效(M1-M4) → 双节点失效(M1+M3)
```
