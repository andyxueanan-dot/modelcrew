## 2. 模型方程

### 2.1 减排
ΔE_i(τ) = E_i × (1 - exp(-ε_i × τ / 100))

剩余排放：E'_i(τ) = E_i - ΔE_i(τ) = E_i × exp(-ε_i × τ / 100)

### 2.2 碳税成本
Cost_i(τ) = τ × E'_i(τ) = τ × E_i × exp(-ε_i × τ / 100)

### 2.3 GDP 损失
ΔGDP_i(τ) = α × Cost_i(τ)

### 2.4 就业损失
劳动生产率：P_i = GDP_i / Job_i  
ΔJob_i(τ) = ΔGDP_i(τ) / P_i = α × Cost_i(τ) × Job_i / GDP_i

### 2.5 碳税收入
Revenue(τ) = Σ Cost_i(τ)

### 2.6 绿色就业
GreenJobs(τ) = β × Revenue(τ) / 10^4 （Revenue 单位：亿元，β 单位：岗/亿元）

### 2.7 净就业
NetJobs(τ) = GreenJobs(τ) - Σ ΔJob_i(τ)

## 3. 关键指标

| 指标 | 公式 | 含义 |
|------|------|------|
| 总减排量 | ΔE(τ) = Σ ΔE_i(τ) | 全省减排 |
| 减排比例 | ΔE(τ) / E_total | 减排效果 |
| 总 GDP 损失 | ΔGDP(τ) = Σ ΔGDP_i(τ) | 经济影响 |
| GDP 损失率 | ΔGDP(τ) / GDP_total | 经济影响比例 |
| 总就业损失 | ΔJob_loss(τ) = Σ ΔJob_i(τ) | 就业影响 |
| 净就业 | NetJobs(τ) | 补贴后净效果 |
| 碳税收入 | Revenue(τ) | 财政收入 |

## 4. 优化目标

### Q2: 碳排放减半
求 τ_min 使得 E_total(τ) ≤ 2.5 亿吨

二分搜索：
- 初始化 τ_low = 0, τ_high = 1000
- 迭代：τ_mid = (τ_low + τ_high) / 2
- 如果 E_total(τ_mid) ≤ 2.5，τ_high = τ_mid
- 否则 τ_low = τ_mid
- 收敛到 τ_min ≈ τ_high

### Q3: 就业中性碳税
求 τ_jobs 使得 NetJobs(τ) = 0

二分搜索：
- 初始化 τ_low = 0, τ_high = 1000
- 迭代：τ_mid = (τ_low + τ_high) / 2
- 如果 NetJobs(τ_mid) ≥ 0，τ_high = τ_mid
- 否则 τ_low = τ_mid
- 收敛到 τ_jobs ≈ τ_high

## 5. 敏感性分析

### 参数空间
- τ ∈ [0, 100, 200, 300, 400, 500]
- 弹性扰动：ε_i × (1 ± 0.2)

### 输出指标
- 减排比例
- GDP 损失率
- 净就业
