# Round 07 — 优化型（第 2 轮）

## 题目：冷链物流中心选址——带容量约束的设施选址问题

某生鲜电商计划在 8 个候选城市中选择若干建设冷链物流中心，服务 12 个区域市场。每个候选城市有建设成本（万元）和最大处理能力（吨/日）。每个区域市场有日均需求量（吨/日）。运输成本与距离成正比（元/吨·公里）。

### 运输成本

距离 = 欧氏距离 × 111 km/度（简化为经纬度直线距离 × 111）  
运输成本 = 2 元/吨·公里

### 约束

1. **需求满足**: 每个市场的需求必须被完全满足
2. **容量约束**: 每个开放的设施的分配量不超过其容量
3. **最少开放数**: 至少开放 3 个设施（保证冗余）
4. **预算约束**: 总建设成本 ≤ 3000 万元

### 问题

**Q1**: 最优的设施选址方案是什么？总成本（建设 + 运输）是多少？

**Q2**: 如果预算从 3000 万降到 2500 万，最优方案如何变化？总成本增加多少？

**Q3**: 如果某个设施（如 F1 北京）因政策原因不能建设，最优方案如何调整？

### 要求

1. 使用完全枚举法（2^8 = 256 种开放组合，筛选可行解）
2. 对每种可行的开放组合，求解运输问题（最小化运输成本）
3. 绘制 Pareto 前沿（总成本 vs 开放设施数）
4. 敏感性分析：预算和运输单价的影响
