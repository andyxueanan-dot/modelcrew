# Model — 设施选址 0-1 整数规划

## 1. 决策变量

- y_j ∈ {0, 1}: 设施 j 是否开放（j = 1..8）
- x_ij ∈ {0, 1}: 市场 i 是否由设施 j 服务（i = 1..12, j = 1..8）

## 2. 参数

| 参数 | 符号 | 值 |
|------|------|-----|
| 设施数 | n | 8 |
| 市场数 | m | 12 |
| 建设成本 | f_j | [800, 900, 750, 650, 600, 700, 550, 600] 万元 |
| 设施容量 | c_j | [500, 600, 550, 450, 400, 500, 350, 350] 吨/日 |
| 市场需求 | d_i | [180, 220, 200, 150, 100, 170, 130, 90, 110, 160, 140, 120] 吨/日 |
| 运输单价 | α | 2 元/吨·公里 |
| 距离 | dist_ij | 欧氏距离 × 111 km |
| 预算 | B | 3000 万元 |
| 最少开放数 | k_min | 3 |

## 3. 目标函数

min Z = Σ f_j · y_j + Σ Σ α · dist_ij · d_i · x_ij

即：最小化（建设成本 + 运输成本）

## 4. 约束

### 4.1 需求满足
Σ_j x_ij = 1, ∀i （每个市场恰好由一个设施服务）

### 4.2 容量约束
Σ_i d_i · x_ij ≤ c_j · y_j, ∀j （开放设施的分配量不超过容量）

### 4.3 预算约束
Σ f_j · y_j ≤ B

### 4.4 最少开放数
Σ y_j ≥ k_min

### 4.5 逻辑约束
x_ij ≤ y_j, ∀i,j （只有开放的设施才能服务市场）

## 5. 求解方法

### 5.1 完全枚举
- 枚举所有 y ∈ {0,1}^8（256 种组合）
- 筛选满足预算 + 最少开放数的组合
- 对每种可行的 y，求解运输子问题

### 5.2 运输子问题（给定 y）
对于开放的设施集合 J = {j : y_j = 1}：
- 将每个市场 i 分配给最近的开放设施 j ∈ J
- 检查容量约束：如果某设施超载，将超额市场重新分配给次近设施
- 计算运输成本：Σ α · dist_ij · d_i

### 5.3 贪心分配算法
```
1. 对每个市场 i，计算到所有开放设施 j 的距离
2. 按距离排序，优先分配给最近设施
3. 如果最近设施容量不足，分配给次近设施
4. 重复直到所有市场分配完毕
```

## 6. 输出指标

| 指标 | 公式 | 含义 |
|------|------|------|
| 总成本 | Z = 建设 + 运输 | 目标函数值 |
| 开放设施数 | Σ y_j | 设施数量 |
| 运输成本 | Σ α · dist_ij · d_i · x_ij | 运营成本 |
| 设施利用率 | Σ d_i · x_ij / c_j | 容量使用比例 |
| 预算使用率 | Σ f_j · y_j / B | 预算消耗比例 |
