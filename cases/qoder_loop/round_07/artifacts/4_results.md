## 2. 预算收紧（Q2，预算 2500 万）

**结果：无可行解。**

原因分析：
- 最便宜的 4 个设施：F7(550) + F5(600) + F8(600) + F4(650) = 2400 万
- 但这 4 个设施的总容量 = 350 + 400 + 350 + 450 = 1550 < 1770（需求）
- 需要加一个更大容量的设施，但 2400 + 700 (F6) = 3100 > 2500
- **预算 2500 万无法同时满足容量和预算约束**

## 3. 设施移除（Q3，F1 不能建设）

**结果：零影响。**

F1（北京）不在最优方案中，移除 F1 后最优方案仍然是 F3+F4+F5+F6，总成本不变。

**解读**：F1 建设成本高（800 万），地理位置偏北，不是最优组合的候选。这说明 F1 在当前参数下是"冗余设施"。

## 4. Pareto 前沿

**结果：退化（只有 k=4 的解可行）。**

原因：
- k=3：容量不足（最便宜的 3 个设施容量 = 1750 < 1770）
- k=4：可行（31 个解）
- k=5+：预算超限（最便宜的 5 个设施 = 3100 > 3000）

**预算约束是紧致的**——这是本题的核心特征。

## 5. 核心发现

1. **预算-容量双重紧致**：预算 3000 万和容量 1770 吨同时约束了可行解空间。只有 4 设施方案可行（31/256 = 12.1%）。

2. **"冗余设施"现象**：F1（北京）和 F2（上海）是两个最贵的设施，都不在最优方案中。它们的高建设成本使其在预算约束下不具竞争力。

3. **地理聚类的有效性**：最优方案的分配体现了明显的地理聚类，运输成本比随机分配低 60%+。

4. **预算敏感性极高**：预算从 3000 降到 2500（-17%），可行解从 31 降到 0。这是组合优化的典型"悬崖效应"。
