# Analysis — 冷链物流中心选址

## 1. 问题分解

### Q1: 最优选址（预算 3000 万）
- **核心**: 枚举所有可行的开放组合，对每种组合求解运输子问题
- **输出**: 最优方案 + 总成本 + 各市场分配

### Q2: 预算收紧（2500 万）
- **核心**: 重新筛选可行解，比较最优方案变化
- **输出**: 新最优方案 + 成本增加幅度

### Q3: 设施缺失（F1 不能建设）
- **核心**: 移除 F1 后重新枚举
- **输出**: 调整方案 + 成本影响

## 2. 假设清单

| # | 假设 | 理由 |
|---|------|------|
| A1 | 运输成本与距离线性（2 元/吨·公里） | 简化模型，不考虑规模经济 |
| A2 | 距离 = 欧氏距离 × 111 km/度 | 简化为直线距离 |
| A3 | 每个市场只能由一个设施服务（单一分配） | 简化模型，不考虑拆分配送 |
| A4 | 设施容量是硬约束（不可超） | 冷链物流中心有物理上限 |
| A5 | 建设成本是一次性投资（年化未考虑） | 简化为静态模型 |

## 3. 约束设计（R1 教训应用）

### 预算约束：3000 万元
- 最便宜的 3 个设施：F7(550) + F5(600) + F8(600) = 1750 万
- 最便宜的 4 个设施：1750 + F4(650) = 2400 万
- 最便宜的 5 个设施：2400 + F6(700) = 3100 万（超预算）
- **预算 3000 万允许最多开放 4 个设施**（如果选便宜的）
- **预算约束接近紧致**（5 个最便宜的 = 3100 万 > 3000 万）

### 容量约束
- 总需求 1770 吨/日
- 如果开放 3 个设施，平均容量需 590 吨/日
- F1-F6 容量 400-600，F7/F8 容量 350
- **容量约束可能紧致**（取决于开放哪些设施）

## 4. 预期陷阱

### 陷阱 1: 运输子问题的复杂度
给定 k 个开放设施和 12 个市场，分配方案有 k^12 种（如果允许拆分则更多）。需要用贪心或 LP 求解。

### 陷阱 2: 地理位置的聚类
华北市场（M1/M7）靠近 F1/F7，华南市场（M3/M9/M12）靠近 F3。最优方案应该体现地理聚类。

### 陷阱 3: 预算-容量的权衡
便宜的设施（F7/F8）容量小（350），贵的设施（F2/F3）容量大（550-600）。预算约束下，可能被迫选择容量小的设施组合。

## 5. 方法选择理由

### 为什么用完全枚举？
- 2^8 = 256 种组合，计算量小
- 可以保证找到全局最优
- 便于分析所有可行解的分布

### 为什么用贪心分配？
- 运输子问题是 NP-hard（如果考虑容量约束）
- 贪心（按距离最近分配）是快速近似
- 可以后续用 LP 优化
