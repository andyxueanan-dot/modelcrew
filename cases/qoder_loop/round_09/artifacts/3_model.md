# Model — 15省高质量发展综合评价

## 1. 数据预处理

### 正向化
- 正向指标 I1, I4, I5, I6: x' = x
- 负向指标 I2, I3: x' = max(x) - x + min(x)

### 标准化 (min-max)
x''_ij = (x'_ij - min_j) / (max_j - min_j) × 0.999 + 0.001

## 2. 熵权法

### 比重矩阵
p_ij = x''_ij / Σ_i x''_ij

### 信息熵
e_j = -k × Σ_i p_ij × ln(p_ij), k = 1/ln(n), n=15

### 冗余度
d_j = 1 - e_j

### 权重
w_j = d_j / Σ d_j

## 3. TOPSIS

### 加权标准化
v_ij = w_j × x''_ij

### 正理想解
A+ = (max_i v_i1, max_i v_i2, ..., max_i v_im)

### 负理想解
A- = (min_i v_i1, min_i v_i2, ..., min_i v_im)

### 距离
D+_i = √(Σ_j (v_ij - A+_j)²)
D-_i = √(Σ_j (v_ij - A-_j)²)

### 贴近度
C_i = D-_i / (D+_i + D-_i)

## 4. AHP (层次分析法)

### 判断矩阵
专家判断：经济 > 创新 > 社会 > 环境

| 维度 | 经济 | 创新 | 社会 | 环境 |
|------|------|------|------|------|
| 经济 | 1 | 2 | 3 | 5 |
| 创新 | 1/2 | 1 | 2 | 3 |
| 社会 | 1/3 | 1/2 | 1 | 2 |
| 环境 | 1/5 | 1/3 | 1/2 | 1 |

### 一致性检验
λ_max = Σ (AW)_i / (n × W_i)
CI = (λ_max - n) / (n - 1)
CR = CI / RI(n=4) = CI / 0.89
要求 CR < 0.1

### 维度内权重分配
- 经济维度: I1 (人均GDP) 权重 1.0
- 社会维度: I2 (基尼) 权重 1.0
- 环境维度: I3 (碳强度) 权重 1.0
- 创新维度: I4 (R&D) 权重 1.0
- 发展维度: I5 (城镇化) 权重 0.5, I6 (HDI) 权重 0.5

（I5 和 I6 归入发展/社会维度，各半）

## 5. 敏感性分析

### 参数扫描
w_env ∈ {0.15, 0.16, ..., 0.35}
w_econ = (1 - w_env) × 0.40
w_social = (1 - w_env) × 0.30
w_innov = (1 - w_env) × 0.30

### 输出
每个 w_env 下的 top-5 排名，标注翻转事件。

## 6. Bootstrap 稳定性检验

### 算法
```
For b = 1 to B=1000:
  1. 从 15 省中有放回抽样 15 个
  2. 对抽样数据计算熵权 + TOPSIS
  3. 记录各省排名 rank_b[i]
  
rank_CI[i] = [percentile(rank[i], 2.5), percentile(rank[i], 97.5)]
```

## 7. 输出指标

| 指标 | 公式 | 含义 |
|------|------|------|
| 熵权 w_j | d_j / Σd_j | 客观信息量权重 |
| 贴近度 C_i | D-/(D+ + D-) | TOPSIS 综合得分 |
| CR | CI/RI | AHP 一致性比率 |
| rank_CI | [P2.5, P97.5] | Bootstrap 排名置信区间 |
