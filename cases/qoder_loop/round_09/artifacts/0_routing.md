# Routing — 15省高质量发展评价

## 题型判定

**类型**: 评价型 (Evaluation)

## 路由理由

1. 多指标综合评价问题，6 个指标涵盖 4 个维度
2. 需要多种 MCDM 方法交叉验证（熵权 TOPSIS + AHP）
3. 含权重敏感性分析和 Bootstrap 稳定性检验
4. n=15 满足统计要求（吸取 R3 n=8 的教训）

## 核心挑战

1. **增长质量悖论**: A 省 GDP 最高但社会/环境指标差，排名取决于权重分配
2. **权重敏感性**: 环境权重变化导致排名翻转，需找临界点
3. **排名稳定性**: Bootstrap 检验排名是否在噪声范围内

## 方法链

```
正向化/标准化 → 熵权法(客观) → TOPSIS排名
              → AHP(主观)     → TOPSIS排名(对比)
              → 敏感性分析(环境权重15%→35%)
              → Bootstrap(B=1000, 95% CI)
```
