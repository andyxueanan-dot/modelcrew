# Analysis — 15省高质量发展评价

## 1. 数据预处理设计

### 正向化
- 正向指标（I1, I4, I5, I6）：直接使用
- 负向指标（I2, I3）：取 max - x 转换为正向

### 标准化
采用 min-max 标准化到 [0, 1]：
x'_ij = (x_ij - min_j) / (max_j - min_j)

为避免熵权法中 ln(0) 问题，平移：
x''_ij = x'_ij × 0.999 + 0.001

## 2. 共线性/相关性预判

| 指标对 | 预期相关性 | 原因 |
|--------|-----------|------|
| I1(人均GDP) - I3(碳强度) | r ≈ +0.8 | 高GDP省份工业密集，碳排放高 |
| I1(人均GDP) - I5(城镇化) | r ≈ +0.9 | 经济发达→城镇化高 |
| I2(基尼) - I3(碳强度) | r ≈ +0.6 | 不平等与污染正相关 |
| I4(R&D) - I6(HDI) | r ≈ +0.5 | 创新投入→人力资本 |
| I5(城镇化) - I6(HDI) | r ≈ +0.7 | 城镇化促进人类发展 |

**陷阱**：I1-I3-I5 高度相关，意味着"经济-环境-城镇化"本质测同一构念（发展水平），熵权法可能给这三个指标高权重（信息量大），导致 A 省和 I 省排名偏高。

## 3. "增长质量悖论"分析

- **A 省**: I1=14.2(第1), I2=0.42(第15,最差), I3=1.85(第14,最差), I5=89.3(第1)
  - 经济维度极强，社会/环境维度极弱
  - 熵权法下可能排名靠前（I1/I5 区分度高→高权重→A 受益）
  
- **B 省**: I1=11.5(第4), I2=0.35(第7), I3=1.20(第9), I4=4.2(第2), I5=76.1(第5), I6=0.85(第1)
  - 无单项第一但全面均衡
  - AHP 下可能排名靠前（专家重视均衡）

- **I 省**: I4=4.5(第1), I5=86.5(第2), I2=0.44(第15,最差), I3=1.92(第15,最差)
  - 创新领先但社会/环境最差
  - 排名高度依赖权重

## 4. 敏感性分析设计

环境权重从 15% 递增到 35%（步长 1%），其他维度权重等比缩放：
- w_env = 15%, 20%, 25%, 30%, 35%
- w_econ = (1 - w_env) × 0.40
- w_social = (1 - w_env) × 0.30
- w_innov = (1 - w_env) × 0.30

关注排名翻转的临界点，特别是 A 省 vs B 省的交叉。

## 5. Bootstrap 设计

B=1000 次重采样，每次从 15 省中有放回抽样 15 个（允许重复），计算 TOPSIS 排名。
输出：每个省份排名的 95% CI（2.5th - 97.5th percentile）。
