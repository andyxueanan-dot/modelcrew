# Round 09 — 评价型（第 2 轮）

## 题目：15 省高质量发展综合评价——增长质量悖论与权重敏感性

某研究机构对 15 个省份的"高质量发展"进行综合评价，涵盖经济、社会、环境、创新四个维度，共 6 个指标。

### 评价指标体系

| 指标 | 符号 | 方向 | 单位 | 维度 |
|------|------|------|------|------|
| 人均 GDP | I1 | + | 万元 | 经济 |
| 基尼系数 | I2 | - | 无量纲 | 社会（负向） |
| 碳排放强度 | I3 | - | 吨CO2/万元GDP | 环境（负向） |
| R&D 支出占比 | I4 | + | % | 创新 |
| 城镇化率 | I5 | + | % | 发展 |
| 人类发展指数 | I6 | + | 无量纲(0-1) | 社会福利 |

### 已知"陷阱"结构

- **省份 A**：人均 GDP 最高（14.2），但基尼系数最高（0.42）、碳排放强度第二高（1.85）——"高增长低质量"
- **省份 I**：R&D 最高（4.5%）、城镇化第二（86.5%），但基尼最高（0.44）、碳强度最高（1.92）——"创新驱动但不可持续"
- **省份 B**：无单项第一，但各指标均衡靠前——"全能型选手"
- **权重敏感点**：经济 vs 环境的权重分配直接决定 A/I vs B/D 的排名翻转

### 问题

**Q1**: 用熵权法确定客观权重，用 TOPSIS 计算 15 省的综合得分和排名。A 省排名第几？

**Q2**: 用 AHP（层次分析法）确定主观权重（假设专家判断经济>创新>社会>环境），与熵权法权重对比。两种权重下的排名是否一致？

**Q3**: 当环境权重从 15% 提高到 35% 时，排名会发生怎样的翻转？找出"翻转临界点"。

**Q4**: 用 Bootstrap 重采样（B=1000）检验排名前 5 的省份的 95% 置信区间。排名是否稳定？

### 要求

1. 数据直接给定（非合成），n=15 满足统计要求
2. 正向化 + 标准化处理（负向指标取倒数或 1-x）
3. 熵权法计算客观权重
4. TOPSIS 计算综合得分
5. AHP 构建判断矩阵 + 一致性检验（CR<0.1）
6. 环境权重敏感性分析（15%→35%，步长 1%）
7. Bootstrap 重采样检验排名稳定性（B=1000，95% CI）
8. 可视化：排名对比图、敏感性曲线、Bootstrap 箱线图
