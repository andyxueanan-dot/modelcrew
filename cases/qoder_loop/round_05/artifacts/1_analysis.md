# Analysis — SIR-V 传染病动力学分析

## 1. 问题分解

### Q1: 无接种基线（v=0）
- **核心**: 求解经典 SIR 模型，得到峰值感染 I_max、峰值时间 t_peak、最终感染比例 R(∞)/N
- **输出**: 时间序列曲线 + 关键指标

### Q2: 接种干预效果（v=5000）
- **核心**: 对比 v=0 和 v=5000 的 I_max 和疫情持续时间
- **输出**: 对比曲线 + 效果量化（峰值下降比例、持续时间缩短天数）

### Q3: 群体免疫阈值
- **核心**: 求使 R_eff < 1 的最低接种速率 v_min
- **公式**: R_eff = R₀ · S_eff/N < 1，其中 S_eff 是考虑接种后的有效易感者
- **输出**: v_min + 达到群体免疫的天数

## 2. 假设清单

| # | 假设 | 理由 |
|---|------|------|
| A1 | 人口封闭（无出生/死亡/迁移） | 疫情周期短（~1 年），人口动态可忽略 |
| A2 | 疫苗有效率 e=0.85（85% 的接种者获得免疫） | 现实疫苗参数 |
| A3 | 接种优先易感者（v·e·S/N 项） | 合理假设：感染者和康复者不接种 |
| A4 | 混合均匀（mass-action 传染项 β·S·I/N） | 经典 SIR 假设 |
| A5 | 康复者永久免疫（无二次感染） | 简化模型 |

## 3. 模型方程

### SIR-V 系统
```
dS/dt = -β·S·I/N - v·e·S/N
dI/dt = β·S·I/N - γ·I
dR/dt = γ·I + v·e·S/N
```

### 守恒律
S + I + R = N（总人口恒定）

### 基本再生数
R₀ = β/γ = 0.3/0.1 = 3.0

### 有效再生数
R_eff(t) = R₀ · S(t)/N · (1 - v·e/(β·I + v·e))

简化：如果接种速率远小于感染速率（v·e << β·I），则 R_eff ≈ R₀ · S(t)/N

## 4. 预期陷阱

### 陷阱 1: 疫苗接种项的归一化
v·e·S/N 表示"每天接种 v 人，有效率 e，优先接种易感者"。但如果 S/N 很小（大部分人已感染或康复），接种速率会下降——这是合理的，但需要注意。

### 陷阱 2: 群体免疫的动态阈值
群体免疫不是静态的（R₀·S/N < 1），而是动态的——随着接种和感染，S 在下降。v_min 应该使 R_eff 在疫情爆发前就 < 1。

### 陷阱 3: 峰值时间的数值精度
odeint 的输出是离散时间点，峰值可能在两个时间点之间。需要用插值或更密的 t_eval 来提高精度。

## 5. 方法选择理由

### 为什么用 odeint？
- 经典 ODE 求解器，稳定可靠
- 自适应步长（LSODA 算法）
- 易于使用（传入函数 + 初值 + 时间点）

### 为什么做敏感性分析？
- R₀ 和 v 是两个关键参数
- 二维热力图能直观展示参数空间
- 帮助决策者理解"接种多少才能控制疫情"

### 为什么做相平面图？
- S vs I 相图能展示系统的动力学特征
- 可以观察"无接种"和"有接种"轨迹的差异
- 直观展示群体免疫阈值（I 开始下降时的 S 值）
