# Model — SIR-V 传染病动力学模型

## 1. 状态变量

- S(t): 易感者数量（人）
- I(t): 感染者数量（人）
- R(t): 康复者数量（人）
- N = S + I + R = 1,000,000（总人口恒定）

## 2. 参数

| 参数 | 符号 | 值 | 单位 |
|------|------|-----|------|
| 总人口 | N | 1,000,000 | 人 |
| 传染率 | β | 0.3 | 每人/天 |
| 康复率 | γ | 0.1 | 每人/天 |
| 基本再生数 | R₀ | 3.0 | 无量纲 |
| 疫苗有效率 | e | 0.85 | 无量纲 |
| 接种速率 | v | 待定 | 人/天 |
| 初始感染者 | I(0) | 100 | 人 |
| 初始易感者 | S(0) | 999,900 | 人 |
| 初始康复者 | R(0) | 0 | 人 |

## 3. ODE 系统

```
dS/dt = -β·S·I/N - v·e·S/N
dI/dt = β·S·I/N - γ·I
dR/dt = γ·I + v·e·S/N
```

### 各项含义
- β·S·I/N：感染传播（mass-action 项）
- v·e·S/N：疫苗接种（优先易感者，有效率 e）
- γ·I：康复

## 4. 关键指标

### 4.1 峰值感染
I_max = max(I(t))  
t_peak = argmax(I(t))

### 4.2 最终感染比例
attack_rate = R(∞) / N

### 4.3 疫情持续时间
duration = 最后一个 I(t) > 1 的时间点

### 4.4 有效再生数
R_eff(t) = R₀ · S(t) / N

群体免疫条件：R_eff < 1，即 S(t) < N / R₀

### 4.5 群体免疫阈值
临界易感者比例：S_c / N = 1 / R₀ = 1/3 ≈ 33.3%

如果通过接种使 S 降到 S_c 以下，则 R_eff < 1，疫情自然消退。

## 5. 群体免疫的最低接种速率

### 静态分析（疫情爆发前）
假设 S(0) ≈ N，要使 R_eff < 1：
S(0) · (1 - v·e·T / N) < N / R₀

其中 T 是接种持续时间。简化：如果接种速率足够快，在疫情爆发前就达到群体免疫：
v_min · e · T_vac = N · (1 - 1/R₀) = N · (1 - 1/3) = 2N/3

如果 T_vac = 30 天（1 个月内完成接种）：
v_min = 2N / (3 · 30 · e) = 2,000,000 / (90 · 0.85) ≈ 26,144 人/天

### 动态分析（考虑疫情同时传播）
更准确的 v_min 需要数值求解：在疫情传播的同时接种，使 I(t) 单调递减（无峰值）。

条件：dI/dt < 0 for all t > 0
即：β·S·I/N - γ·I < 0
即：β·S/N < γ
即：S < N/R₀ = 333,333

初始 S(0) = 999,900 > 333,333，所以初始 dI/dt > 0（疫情会增长）。

要使 I(t) 尽快开始下降，需要快速接种使 S 降到 333,333 以下。

## 6. 敏感性分析

### 参数空间
- R₀ ∈ [1.5, 2.0, 2.5, 3.0, 3.5, 4.0, 4.5, 5.0]
- v ∈ [0, 2000, 5000, 10000, 15000, 20000] 人/天

### 输出指标
- I_max（峰值感染人数）
- t_peak（峰值时间）
- attack_rate（最终感染比例）

## 7. 输出指标汇总

| 指标 | 公式 | 含义 |
|------|------|------|
| I_max | max(I(t)) | 峰值感染人数 |
| t_peak | argmax(I(t)) | 峰值时间（天） |
| attack_rate | R(∞)/N | 最终感染比例 |
| duration | last t where I(t)>1 | 疫情持续时间 |
| R_eff(t) | R₀·S(t)/N | 有效再生数 |
| v_min | numerical search | 最低接种速率 |
