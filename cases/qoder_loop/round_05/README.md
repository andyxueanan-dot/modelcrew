# Round 05 — 机理ODE型

## 题目：疫苗接种干预下的传染病传播动力学

某城市 100 万人口，爆发一种新型呼吸道传染病。疾病传播符合 SIR 模型（易感者 S、感染者 I、康复者 R），但引入疫苗接种作为干预手段。

### 模型参数

**基础参数**：
- 总人口 N = 1,000,000
- 初始感染者 I(0) = 100
- 初始康复者 R(0) = 0
- 初始易感者 S(0) = N - I(0) - R(0) = 999,900
- 传染率 β = 0.3（每人/天）
- 康复率 γ = 0.1（每人/天）
- 基本再生数 R₀ = β/γ = 3.0

**疫苗接种参数**：
- 疫苗有效率 e = 0.85（85%）
- 接种速率 v（待定，单位：人/天）
- 接种覆盖目标：60% 人口（600,000 人）

### SIR-V 模型方程

```
dS/dt = -β·S·I/N - v·e·S/N
dI/dt = β·S·I/N - γ·I
dR/dt = γ·I + v·e·S/N
```

其中：
- β·S·I/N：感染传播项
- v·e·S/N：疫苗接种项（假设优先接种易感者）
- γ·I：康复项

### 问题

**Q1**：无疫苗接种（v=0）时，疫情的峰值感染人数、峰值时间、最终感染比例是多少？

**Q2**：如果以 v = 5000 人/天的速率接种，峰值感染人数下降多少？疫情持续时间缩短多少？

**Q3**：要达到"群体免疫"（R₀ < 1），最低接种速率 v_min 是多少？此时需要多少天达到群体免疫？

### 要求

1. 使用 scipy.integrate.odeint 求解 ODE 系统
2. 绘制 S/I/R 随时间变化的曲线图（无接种 vs 有接种）
3. 绘制相平面图（S vs I）
4. 敏感性分析：R₀ 和 v 对峰值感染人数的影响
5. 计算群体免疫阈值和最低接种速率
