## 情景

某省级主管部门要对下辖 **10 个地区（A–J）** 的 **"绿色低碳发展水平"** 做一次综合评价，用于资源倾斜与考核排名。已采集到每个地区在 **7 项指标** 上的最新统计值（见"给定数据"），现需要一套**客观、可解释、可复现**的评价方法，给出排名并支撑决策。

## 给定数据

数据文件：`data/indicators.csv`（10 个地区 × 7 项指标），各指标含义与**方向**如下：

| 指标列名 | 含义 | 单位 | 方向 |
|---|---|---|---|
| `energy_intensity` | 单位 GDP 能耗 | 吨标煤/万元 | **逆向**（越小越好） |
| `renewable_share` | 可再生能源发电占比 | % | 正向（越大越好） |
| `forest_cover` | 森林覆盖率 | % | 正向 |
| `carbon_intensity` | 单位 GDP 碳排放 | 吨 CO₂/万元 | **逆向** |
| `solid_waste_util` | 一般工业固废综合利用率 | % | 正向 |
| `park_green_percap` | 人均公园绿地面积 | m² | 正向 |
| `pm25` | PM2.5 年均浓度 | μg/m³ | **逆向** |

> 数据为该轮评价的**唯一给定输入**，无其他外部数据；数值结果应由求解脚本**确定性**生成（无随机或固定随机种子），保证字节级可复现。

## 要求

- **Q1（核心）**：建立一套综合评价模型，对 A–J 十个地区的"绿色低碳发展水平"**排序**，给出完整排名、各地区综合得分及排序依据。须说明指标如何统一方向与量纲、权重如何确定。
- **Q2（诊断与建议）**：指出**排名靠后**地区的**主要短板维度**（是哪些指标拖累），并据此给出**有针对性、可执行**的提升建议（避免"多搞环保"这类空泛说法）。
- **Q3（稳健性与局限）**：你的排序结论对**赋权方法的选择、指标口径、数据扰动**有多敏感、多可靠？如实评估方法的适用边界与局限（例如：换一种权重方案，排名会不会大幅翻动？哪些地区的名次最不稳？）。
