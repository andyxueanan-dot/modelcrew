# 练习题 01 · 月度销量预测（预测型 / 时间序列）

> ModelCrew 自练案例（`/loop` 刷题）。题型：**预测型（C）**，方法箱：时间序列（指数平滑 / ARIMA / 趋势+季节回归）。
> 选它因为现有 4 案例（体育统计 / 金融 ML / 选址优化 / 因果混杂）都**没用过时序方法箱**，且预测题天然考"**留出验证**"——获奖论文的硬门槛（§3A-5 模型验证）。

## 题面

某零售商品有连续 48 个月的销量记录（`data/sales.csv`，字段 `month`(1..48)、`sales`）。数据带**上升趋势 + 年度季节性 + 噪声**。要求：

- **Q1**：刻画该序列的趋势与季节结构，选合适模型拟合。
- **Q2**：预测未来 12 个月销量，并给出**预测不确定性**（区间）。
- **Q3**：你的模型可靠吗？如何验证不是过拟合？给出诚实的误差评估与改进方向。

## 实质

不是"套一个模型出一条曲线"，而是：**先建立对趋势/季节的认识 → 由简到繁选模型并用留出集验证 → 报带不确定性的预测 + 诚实误差**。考点正是获奖论文的"模型验证 + 不确定性 + 由简到繁"。

数据 `data/sales.csv` 由 `artifacts/solve.py` 确定性生成（seed=42），字节级可复现。
