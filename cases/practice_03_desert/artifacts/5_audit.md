## 四、对抗辩论（高风险结论：「挖矿不划算」是真是假）

- **理论家（主张 9410 直达最优、挖矿假象不存在）**：直达是**已验证可行解**（手算 9410）；挖矿要绕 ≥13 步进出矿 + k 挖矿日，在 1200kg 下一次背不动、进村按 2 倍价补给，独立穷举全路径上界仅 8435 < 9410。三条证据（手算下界 + 全路径上界 + A=B 互证）闭环。
- **怀疑者（主张挖矿被漏搜、9410 是假最优）**：solver 只试固定 BFS 路线，可能漏掉某条"刚好负重可行又挖得多"的刁钻路线；全规模带负重 DP 纯 Python 跑不完，**没有**真正的全状态暴力背书。
- **裁判（我）**：怀疑者的攻击点真实存在——solver 确实没做全状态暴力。**但我用独立的全简单路径穷举（230×358×k，远超 solver 的固定 3 条路线）顶上了这个缺口，结果仍是 8435**。怀疑者的"漏搜"假设被我的更广搜索证伪。全规模 DP 跑不完不影响结论：因为(a)直达 9410 是确定可行下界，(b)我的全路径挖矿上界 8435 < 9410 覆盖了唯一能赚钱的动作（挖矿）的所有路线族，(c)不挖矿则不可能超过"恰买够直达"的 9410（多走/多囤只增消耗与净亏）。→ **裁定「挖矿不划算、9410 最优」✅ 站得住**，但最优性属于"强证据支持"级别而非"全状态暴力证明"级别，论文措辞应为"由全路径精确枚举 + 双范式互证支持的最优"，**不要**写"已被全状态穷举证明"。
