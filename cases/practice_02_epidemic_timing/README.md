# 练习题 02 · 传染病干预时机

> ModelCrew 自练案例（`/loop` 刷题 iter2）。**本 README 仅为问题陈述**——解法、模型、创新点全部由 clean-context 的 modelcrew 各角色子代理从零产出，编排者不注入任何解题内容（基准纯净性，见 `cases/practice_log.md`）。
> 题型登记（仅供归档，不构成解法提示）：机理/连续型。

## 情景

某地暴发一种通过人际接触传播的传染病。已知与给定参数：

- 总人口 **N = 1,000,000**，初始感染者 **I₀ = 10**，其余为易感者。
- 平均传染期约 **6 天**（即恢复率 γ ≈ 1/6 /天）。
- 该病的**基本再生数 R₀ = 3.0**。
- 卫生系统对**同时在治**感染者的承载上限为总人口的 **h = 8%**（超过即视为挤兑）。
- 当局可在第 **τ** 天起实施社交干预，使有效接触率（传播率 β）**降低比例 c** 并一直维持；c 与 τ 是可调的政策变量。

## 要求

- **Q1**：建立可刻画该传染病传播的数学模型，并据给定参数刻画**无干预**情形：基本再生数、同时在治感染者的峰值规模与出现时点、最终累计感染比例。
- **Q2**：引入"第 τ 天起把接触率降低 c"的干预。在合理选定的干预力度下，求**能使峰值同时在治感染不超过容量 h 的干预时机**；并就关键参数存在不确定性这一现实，给出**稳健的**时机/力度建议。
- **Q3**：你的结论对关键参数（R₀、c、h 等）有多敏感、多可靠？如实评估模型的局限与适用边界。
