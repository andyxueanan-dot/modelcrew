## 1. 基线模型：确定性 SIR（Q1）

### 1.1 状态方程（mass-action，闭合人群）

$$
\frac{dS}{dt} = -\beta\,\frac{S\,I}{N},\qquad
\frac{dI}{dt} = \beta\,\frac{S\,I}{N} - \gamma\,I,\qquad
\frac{dR}{dt} = \gamma\,I
$$

- 不变量：`S + I + R = N`（恒定，用于求解器守恒性自检）。
- 状态变量：`S(t)` 易感、`I(t)` 同时在治（= prevalence，**容量约束作用对象**）、`R(t)` 移除。
- 初值：`S0 = N - I0 = 999990`，`I0 = 10`，`R0_state = 0`。

### 1.2 参数标定

| 参数 | 值 | 来源/推导 |
|---|---|---|
| N | 1,000,000 | 题给 |
| I0 | 10 | 题给 |
| γ | 1/6 /天 | 平均传染期 6 天 |
| 𝓡₀ | 3.0 | 题给 |
| **β** | 𝓡₀·γ = 3·(1/6) = **0.5 /天** | 由 𝓡₀=β/γ 反推 |
| h（容量，绝对） | 0.08·N = **80,000** 人 | 8% N，硬阈值 |

### 1.3 阈值量（无量纲分析，用作 Solver 数值核对的解析对照）

- **基本再生数**：`𝓡₀ = β/γ`。Q1-b 要求自洽推导而非照抄——线性化 `dI/dt|_{S≈N} = (β−γ)I`，初期指数增长率 `r = β−γ`，每代次再生 `β/γ`，故 `𝓡₀ = β/γ = 3.0`。✔
- **峰值的解析判据（Q1-c 的对照）**：`dI/dt = 0` ⟺ `S = γN/β = N/𝓡₀`。即 **I 峰出现在 S 降到 `N/𝓡₀` 那一刻**（这是检验数值峰值的硬性解析锚点）。
- **峰值规模解析式**（SIR 守恒积分，初期 R≈0、S0≈N）：
  $$
  I_{\max} = N\left(1 - \frac{1}{\mathcal R_0} - \frac{\ln \mathcal R_0}{\mathcal R_0}\right)
  $$
  代入 𝓡₀=3：`I_max/N = 1 − 1/3 − ln3/3 ≈ 1 − 0.3333 − 0.3662 = 0.3005`，即 **≈ 300,500 人**（粗解析，Solver 用数值积分给精确值与 `t_peak`）。→ 远超 h=80,000，**无干预必挤兑**（落实 T5）。
- **最终规模方程（Q1-d）**：超越方程
  $$
  z = 1 - e^{-\mathcal R_0\, z},\qquad z = 1 - S(\infty)/N
  $$
  𝓡₀=3 时数值解 `z ≈ 0.9405`（**≈ 94% 累计感染**）。Solver 用根求解 + ODE 长时积分双向核对。

> **Q1 三个必答数（Solver 算精确值，上面是解析锚点）**：`I_max ≈ 0.30N`、`t_peak`（待 Solver）、`z ≈ 0.94`。

## 2. 干预模型：阶跃降 β（Q2）

### 2.1 干预如何进入方程

干预 = 在 `t = τ` 把传播率从 β 阶跃降到 `β(1−c)` 并永久维持：
$$
\beta(t) = \begin{cases} \beta_0 = 0.5 & t < \tau \\ \beta_0(1-c) & t \ge \tau \end{cases}
$$
方程结构不变，仅 β 变为分段常数。`c ∈ (0,1)` 为干预力度，`τ ≥ 0` 为干预起始日。

- 干预后**瞬时**有效再生数（未计 S 耗损）：`R_e = 𝓡₀(1−c)`。
- **关键阈值（落实 T3）**：`c* = 1 − 1/𝓡₀ = 1 − 1/3 ≈ 0.667`。
  - `c ≥ c*` ⟹ `R_e ≤ 1`，干预后 I 立即（在当前 S 下）转降，峰值至多停在 τ 时刻的 I。
  - `c < c*` ⟹ 干预后 I 仍可继续上升，峰值落在 τ 之后，是否达标取决于 τ 早晚——τ 与 c **耦合**。

### 2.2 可行性判据（核心交付）

定义**峰值泛函**：给定政策 `(τ, c)` 与参数 `θ = (𝓡₀, γ, N, I0)`，
$$
P(\tau, c;\theta) \;=\; \max_{t\ge 0} I(t \mid \tau, c, \theta)
$$
**达标（不挤兑）判据**：
$$
g(\tau, c;\theta) \;=\; P(\tau, c;\theta) - h \;\le\; 0
$$

### 2.3 Q2-a 选定干预力度 c（给出区间而非单点，便于讨论）

按 T3 围绕 `c*≈0.667` 取一组代表力度，正式选定 **基线 c = 0.6**（略低于 c*，刻意落在"干预后仍增长、必须靠时机"的难区，使 τ* 问题非平凡且有现实意义——现实中很难做到 R_e≤1）。并对 **c ∈ {0.4, 0.5, 0.6, 0.7}** 全扫做可行域。
- 理由：c=0.7>c* 会让问题退化为"几乎任何不太晚的 τ 都行"，失去"求时机"的张力；c=0.6 既现实又能凸显 τ* 的存在性与敏感性。

### 2.4 Q2-b 最迟可行干预日 τ*（given c）

**单调性论断（待 Solver 数值验证，列入审计清单 C7）**：固定 c，`P(τ,c)` 关于 τ **单调不减**（干预越晚，S 已耗损越多、I 已积累越高，峰值越高）。故存在唯一临界
$$
\boxed{\;\tau^*(c) = \sup\{\tau \ge 0 : P(\tau, c;\theta) \le h\}\;}
$$
即满足 `g(τ*, c) = 0` 的根。求法：对 τ 做**二分法**（依赖单调性）或细网格扫描 + 线性插值定根。`τ < τ*` 可行，`τ > τ*` 挤兑。
- 若 `P(0,c) > h`（即第 0 天干预都压不住），则在该 c 下**无可行 τ**，诚实报告（落实 T5），不硬凑。

### 2.5 Q2 加分：可行域

在 `(τ, c)` 平面画 `g(τ,c)=0` 等高线，分隔可行区（左下，早干预/强干预）与挤兑区（右上）。给出 `P(τ,c)` 峰值曲面。

## 3. 稳健性与敏感性（Q2-c / Q3）

### 3.1 参数不确定区间（回应 T7"约/≈"）

| 参数 | 标称 | 不确定区间 | 依据 |
|---|---|---|---|
| 𝓡₀ | 3.0 | [2.5, 3.5] | "R₀≈3"，±0.5 |
| γ | 1/6 | 1/[5,7] 即 [0.143, 0.20] | "约 6 天" → 5–7 天 |
| c（执行折扣） | 0.6 | 实际 [0.7c, c]（依从性≤100%） | A5 执行力度有偏差 |
| h | 80,000 | 单边敏感（若仅床位子集则更宽松） | A7 口径待核实 |

### 3.2 稳健干预建议（命名指标，见 §5）

**最坏情形设计**：在不确定盒 `Θ = [𝓡₀]×[γ]×[c_eff]` 上取使峰值最大的角点（直觉：𝓡₀ 取上端、c 取折扣下端、γ 取使 𝓡₀ 维持高的方向），求
$$
\tau^*_{\text{robust}}(c) = \sup\{\tau : \max_{\theta\in\Theta} P(\tau,c;\theta) \le h\}
$$
再相对标称 `τ*` 给出**安全提前量** `Δτ = τ* − τ*_robust`（应提前多少天才稳）。

**安全裕度口径**：另给"峰值留 20% 余量"版本（判据 `P ≤ 0.8h`），作为运营建议下限。

### 3.3 Q3-a 敏感性分析

- **单参数扫描**：分别扫 𝓡₀∈[2.5,3.5]、c∈[0.4,0.7]、h∈[6%,10%]N，画 `τ*` 对各参数的响应曲线。
- **局部弹性**：在标称点数值差分求 `∂τ*/∂𝓡₀` 等，报告无量纲弹性 `(p/τ*)(∂τ*/∂p)`，量化"哪个参数最致命"。预判 𝓡₀ 最敏感（T4：早期指数增长 `e^{(β−γ)t}`，𝓡₀ 小幅上调使达标窗口显著左移）。
- （加分）Sobol 全局敏感性指数，若时间允许。

### 3.4 Q3-b 局限与适用边界

直接继承 Analyst A1–A8：无潜伏期（SEIR 对照会更晚更平 → SIR 的 τ* 偏激进/保守需说明）、均匀混合（低估早期增速）、闭合人群单波（免疫衰减会多波使单峰分析失效）、阶跃干预（高估即时与持久效果 → 实际需更早）、c 完美执行（依从性<100%）。适用范围：单波、数月尺度、无显著年龄/空间结构的人传人急性传染病。

## 4. 数值与可复现规范（落实 A8）

- 求解器：`scipy.integrate.solve_ivp`，方法 `Radau` 或 `LSODA`（SIR 在 R₀=3 接近峰值时偏刚性），`rtol=1e-8, atol=1e-8`，`dense_output=True` 取峰值。
- 峰值定位：先粗积分定区间，再用 `dI/dt=0` ⟺ `S=N/𝓡₀(t)` 的事件函数（`solve_ivp` event）精确定 `t_peak`。
- 最终规模：长时积分（t 到 I<1e-3）的 `1−S/N` 与最终规模方程根求解（`scipy.optimize.brentq`）双向核对，差 <1e-3 才放行。
- τ* 二分：`brentq` 求 `g(τ,c)=0`，区间 `[0, τ_no_intervention_peak]`。
- 固定上述设置 → 字节级可复现。无随机性。

## 5. 创新点 / 命名

- **「SIR 三相干预相图」（Intervention Phase Diagram）**：在 `(τ, c)` 平面用 `c*=1−1/𝓡₀` 与 `g(τ,c)=0` 把政策空间切成三相——
  (i) **强干预相** `c≥c*`：R_e≤1，τ 宽松；
  (ii) **时机敏感相** `c<c*` 且 `τ<τ*(c)`：靠抢时间达标；
  (iii) **失守相** `τ>τ*(c)`：无论如何挤兑。
- **「安全提前量 Δτ」（Robust Lead-Time）**：稳健 τ* 相对标称 τ* 需提前的天数，作为给决策者的**单一可执行数字**——回答"参数不确定下，到底该早几天动手"。
